◆ ZwHistoryUserAttributeGet()

ZW_API_C ezwErrors ZwHistoryUserAttributeGet ( szwEntityHandle  historyHandle,
const zwString64  attributeName,
int *  count,
szwUserAttribute **  attributeList 
)

#include <zwapi_attribute.h>

Outputs the specified user attributes data if attributeName is not NULL, else get all,
of the specified feature or sketch in the activated part.
Function Guide: History Entity(right click) -> Attributes -> User Attribute

Note
The caller is responsible to free the output data by using ZwUserAttributeDataFree()
See also
ZwUserAttributeDataFree, ZwHistoryUserAttributeSet and ZwHistoryUserAttributeDelete
Parameters
[in]historyHandlethe history(just support feature and sketch) handle
[in]attributeNamename of the user attribute, if NULL get all attribute
[out]countthe whole number of user attributes in the specified component.
[out]attributeListdata list of user attributes
Return values
succeeds
  • ZW_API_NO_ERROR
fail
  • ZW_API_GENERAL_ERROR
  • ZW_API_ROOT_OBJ_ACT_FAIL
  • ZW_API_INVALID_INPUT
  • ZW_API_INVALID_OUTPUT
  • ZW_API_INPUT_TYPE_ERROR
  • ZW_API_OBJ_DATA_GET_ERROR
  • ZW_API_MEMORY_ERROR
