◆ cvxLicModulesGet()

ZW_API_C evxErrors cvxLicModulesGet ( char  szModules[1024])

#include <zwapi_license.h>

Get current authorized module list, the "szModules" is split by '
', e.g. "ZW3D_BASE_MODULES\nZW3D_2AXIS_TURNING\nZW3D_3AXIS_TURNING". If fail to check current license, the license is trial(szModules will be set to "Trial").

Parameters
[out]szModules[1024]the module name list to be return.(e.g. "ZW3D_BASE_MODULES\nZW3D_2AXIS_TURNING\nZW3D_3AXIS_TURNING").
Return values
succeeds
  • ZW_API_NO_ERROR
fail
