ZW_API_C ezwErrors ZwMemoryFree ( void **  memoryPointer)

#include <zwapi_memory.h>

Frees memory at specified pointer. Sets pointer to NULL if memory was freed successfully. If the input pointer is already NULL, this function does nothing.

Parameters
[in]memoryPointeraddress of pointer to be freed
Return values
succeeds
  • ZW_API_NO_ERROR
fail
  • ZW_API_GENERAL_ERROR
  • ZW_API_INVALID_INPUT
