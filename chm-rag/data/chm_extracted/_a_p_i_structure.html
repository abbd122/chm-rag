API Structure Introduction

API Structure

1. Basic Modules

General modules including general interface, structure, enumeration, ect.

1.1 General Data

General interface, structure, and enumeration. Define header file in zwapi_util.h.

1.2 Configuration

Corresponding to function of configuration in ZW3D, this can get or modifiy the current configuration. The default configuration locates at install directory:supp/config.xml, and name is corresponding to the first input parameter and header file defined in zwapi_config.h

1.3 Global Application

Global settings or global tools, sucn as configuration of unit, color, software version, language, sketch grid, etc. And tools of inquiring name and label matching. Define header file in zwapi_global_apply.h

1.4 Authorization

Authorization related API. Define header file in zwapi_license.h

1.5 Display

Software display area related API. Display on/off, light and scale, etc. Define header file in zwapi_display.h

1.6 Message

Message processing module, such as printing different type of error and warning message. Define head file in zwapi_message.h

1.7 View

General view API. Define header file in zwapi_view.h

1.8 Memory

Memory released API. Define header file in zwapi_memory.h

1.9 File Root

File root related API, and general interface for single file and multi-root file. Define header file in zwapi_root.h

1.10 Layer

Layer general API. Define header file in zwapi_layer.h

1.11 Event

Event related API, such as undo and redo, ect. Define header file in zwapi_xn.h

1.12 Standard General API

Dimension general API, such as operation of dimension attribute, dimension text and value, etc. Define header file in zwapi_dim.h
Creating dimension interface is in "Command Creation" module, which classifies in different environments and ZW3D Ribbon.
In different environments, API about dimension inquire is in "Environment Object" module.

1.13 Expression General API

Expression general API and expression, etc. Define header file in zwapi_var.h
In different environments, API about expression inquire is in "Environment Object" module.

1.14 Text General API

Text general API, such as operation of dimension attribute, dimension text and value, etc. Define header file in zwapi_text.h
Creating text interface is in "Command Creation" module, which classifies in different environments and ZW3D Ribbon.
In different environments, API about text inquire is in "Environment Object" module.

1.15 Entity General API

All object general API, such as inquiring object name, id, and setting color, etc. Define header file in zwapi_general_ent.h
In general, these history objects are created by modeling commands in "Command Creation" module.

1.16 User Interaction Module

Interaction API of user pick and selection. Define header file in zwapi_userinput.h

3.3 Face API

3.6 Vertex API
