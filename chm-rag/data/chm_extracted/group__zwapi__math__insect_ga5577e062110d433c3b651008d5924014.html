◆ cvxIsectFaceFace()

ZW_API_C evxErrors cvxIsectFaceFace ( int  idFace1,
int  idFace2,
evxFaceTrim  Trim,
int  TangentOk,
int *  Count,
svxCurve **  Curves 
)

#include <zwapi_math_insect.h>

Outputs a list of curves where the two input faces intersect.
the output Curves data can be added to the active part with the cvxPartCurve() function.

The calling procedure MUST deallocate the internal data in each element of the output list by calling cvxCurveFree(), after which the output list MUST be deallocated using cvxMemFree((void**)Curves).

Parameters
[in]idFace1id of the first face entity in active file
[in]idFace2id of the second face entity in active file
[in]Trimface trim-curve option
[in]TangentOk1 to include curves of tangency; else 0
[out]Countnumber of intersection curves
[out]Curveslist of intersection curves
Return values
succeeds
  • ZW_API_NO_ERROR - one or more intersection curves are found
fail
  • ZW_API_GENERAL_ERROR - function fails or no intersection curves is found.
  • ZW_API_INVALID_INPUT
  • ZW_API_INVALID_OUTPUT
  • ZW_API_ROOT_OBJ_ACT_FAIL
  • ZW_API_MODELING_DATA_ERROR
  • ZW_API_MEMORY_ERROR
