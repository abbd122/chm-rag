ZW_API_C ezwErrors ZwDisplayState ( ezwDisplayOption  option,
zwDisplayState  state 
)

#include <zwapi_display.h>

Disable or re-enable display and UI update status. The current display settings are saved to variable "state" when display are disabled, and the display settings in variable "state" are restored when display are re-enabled.

Note
If the function is being used to re-enable display/UI update after an error occurred in application code, always use "option = ZW_DISPLAY_ENABLE_WITHOUT_REFRESH".
In this case "option = ZW_DISPLAY_ENABLE_REFRESH" could cause further errors by attempting to refresh the display from an erroneous database.
