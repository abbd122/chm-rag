ZW_API_C ezwErrors ZwReferenceCreate ( szwEntityHandle  geometry,
ezwReferenceCopyOption  associativeCopy,
szwEntityHandle *  handle 
)

#include <zwapi_dataexchange.h>

Creates a copy of the specified external geometry (face, shape, curve or edge) in the active target part. The input geometry handle must give the full traversal path to the external geometry relative to the top-level part of the parent assembly of the active target part. The handle of the new geometry created in the active part is output via "handle". Function Guide: Data Exchange->Import->Reference

Note
The caller is responsible to free the handle by using ZwEntityHandleFree().
Parameters
[in]geometryhandle of external geometry relative to top-level parent file
[in]associativeCopyreference copy option
[out]handlehandle of new geometry (NULL to ignore)
