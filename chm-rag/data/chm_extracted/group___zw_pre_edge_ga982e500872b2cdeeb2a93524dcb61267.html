#include <zwapi_preedge.h>

Evaluates a pre-edge at a specified parameter value. Computes directional and point on the surface from the UV curve.

Note
"level" specifies the level of evaluation. The higher the level, more data gets returned (at the expense of greater computation).
                     level
      |  0  |       1        |       2        |     3
      +-----+----------------+----------------+-------------
      | pnt |  derivative_1  |  derivative_2  | derivative_3
Parameters
[in]preedgeHandlehandle of preedge
[in]tt parameter value on curve
[in]levellevel of evaluation(0 ~ 3). See description below
[out]evaluationresult
Return values
succeeds
  • ZW_API_NO_ERROR
fail
  • ZW_API_GENERAL_ERROR
  • ZW_API_INVALID_INPUT
  • ZW_API_INVALID_OUTPUT
  • ZW_API_MODELING_DATA_ERROR
  • ZW_API_OBJ_DATA_GET_ERROR
  • ZW_API_MEMORY_ERROR
