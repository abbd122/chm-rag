Detailed Description

reference entity data.

fileName,partName and entClass parameters of this structure must be filled in, and other parameters are optional according to whether the referenced part or assembly.

if entClass=0,compPath and depth must be entered. if entClass>0,idShape or shellName must be entered and idShape takes precedence over shellName, if idShape > 0, shellName is ignored.
