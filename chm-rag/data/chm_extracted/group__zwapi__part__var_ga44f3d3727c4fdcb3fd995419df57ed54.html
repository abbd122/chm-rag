ZW_API_C evxErrors cvxPartEqnMoveToEqnSet ( int  idEqnSet,
int  cnt,
const int *  idListEqn

#include <zwapi_part_var.h>

Moves some existing expressions to the specified equation set.

Unlike the cvxPartVarAddToEqnSet() interface, this interface changes
the position of original expression.

The cvxPartEqnSet() interface can create a equation set and this interface can not.
