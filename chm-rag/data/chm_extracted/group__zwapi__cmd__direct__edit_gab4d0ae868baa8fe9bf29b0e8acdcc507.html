ZW_API_C evxErrors cvxPartDEPtn ( const svxDEPattern *  data,

#include <zwapi_cmd_direct_edit.h>

Use this function to DE(Direct Edit) pattern faces. Six different methods of pattern are available, each method requires different types of input.

Note
Interface cvxPartDEPtnInit() can initialize variables DE(Direct Edit) pattern faces of this function.

[in]datapart or assembly DE(Direct Edit) pattern input data
[out]idPtnid of DE(Direct Edit) pattern feature (NULL to ignore)
