ZW_API_C ezwErrors ZwCommandSend ( const char *  command)

#include <zwapi_command.h>

Executes the input command synchronously.

If you registers a pointer to a "void MyFunction(void)" function by using ZwCommandFunctionLoad(), you can invoke this command by this function with the parameter 'command' passes by "~MyFunction".

Note
The parameter 'command' can't be null pointer, the function ZwCommandPostExecuteAll() do the same thing as 'cvxCmdSend(NULL)'.
This function will execute the command on the current thread.
Parameters
[in]commandnull-terminated ascii command string
Return values
succeeds
  • ZW_API_NO_ERROR
fail
  • ZW_API_GENERAL_ERROR
  • ZW_API_INVALID_STRING
// if you registered a command named 'MyFunction', it will be executed immediately
ZwCommandSend("~MyFunction");
