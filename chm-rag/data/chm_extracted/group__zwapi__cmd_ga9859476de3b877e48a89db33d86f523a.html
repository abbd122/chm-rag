◆ cvxMacroInqPath()

ZW_API_C evxErrors cvxMacroInqPath ( vxPath  path,
vxPath  file 
)

#include <zwapi_cmd.h>

Gets the path and file name of the currently running macro. If no running macro, path[0] = 0 and file[0] = 0. The path and file cannot both be NULL.
