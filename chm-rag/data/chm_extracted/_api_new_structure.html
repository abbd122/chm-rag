New API Structure Introduction

New API Structure

Remarkï¼šThis is the new version of the ZW3D interface, which has a more complete naming and clearer module division compared to the old interface, making it convenient for you to quickly refer to it.New features of the new API:

  1. Reduce the use of interface name abbreviations, align interface names with ZW3D platform features, and facilitate your quick retrieval.
    2.Using handles to manipulate objects, making them more accurate and stable. Note that the handles needs to be released after use.
    3.The default environment for interfaces now is the part environment and also can be used in assembly environment, without labeling of "Part".
    4.The new interface is still being converted. You can use the conversion interface to use both new and old interfaces simultaneously. After all interfaces has been converted, It is recommended that you use the new interface.Some old interfaces will no longer be maintained due to mechanisms and other reasons.
    We will continue to supplement and upgrade the interface.
