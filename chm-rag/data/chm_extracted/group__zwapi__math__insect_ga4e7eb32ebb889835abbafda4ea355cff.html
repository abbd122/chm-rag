ZW_API_C int cvxIsectRayPlane ( svxAxis *  Ray,
svxMatrix *  Plane,
svxPoint *  Point 
)

#include <zwapi_math_insect.h>

Intersects the input ray with the XY plane of the specified transformation matrix and outputs the point of intersection.

Parameters
[in]Rayray defined by a point and a direction
[in]Planeplane transformation matrix (XY plane)
[out]Pointintersection point
Return values
succeeds
  • ZW_API_NO_ERROR - intersection point successfully found.
  • 1 - if the ray is parallel to, and on the plane.
  • 2 - if the ray is parallel to, but off the plane.
fail
  • ZW_API_GENERAL_ERROR
  • ZW_API_INVALID_INPUT
  • ZW_API_INVALID_OUTPUT
