ZW_API_C ezwErrors ZwActiveLCSGet ( szwEntityHandle *  handle)

#include <zwapi_datum.h>

Get the entity handle of active LCS(local coordinate system). If the LCS is equal to the Default CSYS, handle will be set as the handle of Default CSYS.

szwEntityHandle handle;
ZwActiveLCSGet(&handle);
...
ZwEntityHandleFree(&handle);
