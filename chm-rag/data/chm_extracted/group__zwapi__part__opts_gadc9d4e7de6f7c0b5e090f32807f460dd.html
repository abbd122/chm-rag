◆ cvxPartHistRename()

ZW_API_C evxErrors cvxPartHistRename ( int  idOp,
vxName  Name,
char *  Descript 
)

#include <zwapi_part_opts.h>

Renames an operation in the history of the active part and assigns the specified description to it (if defined). Passing in a blank string for "Descript" will clear the description currently assigned to "idOp".

Parameters
[in]idOpid of history operation (i.e. feature)
[in]Namenew operation name
[in]Descriptnew description (NULL if undefined)
