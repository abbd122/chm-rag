◆ cvxCmdSend()

ZW_API_C void cvxCmdSend ( char *  CommandString)

#include <zwapi_cmd.h>

Executes the input command string. If "String = NULL", the event queue is polled for buffered command strings, which are allowed to execute in the order they were buffered. When this function is called, any current thread of execution is suspended until pending command strings are processed, after which this function returns control to the calling procedure.

Deprecated:
This API will not be updated anymore, please use ZwCommandSend()
