#include <zwapi_datum.h>

Create a datum axis object by a cylinder or cone face or arc curve.

Note
The memory referenced by "axis" is allocated by this function. The calling procedure is responsible to free it (ZwEntityHandleFree(&axis)).
Parameters
[in]geometrya cylinder or cone face or arc curve
[in]flipflip the direction of the axis, 0-keep default, 1-flip
[out]axisaxis handle (NULL to ignore)
