◆ cvxGetPoint()

ZW_API_C evxErrors cvxGetPoint ( const char *  Prompt,
evxPntInpOpt  Opt,
int  EmptyOk,
svxPoint *  Point,
int *  idEntity 
)

#include <zwapi_userinput.h>

Displays the specified "prompt" and waits for the user to input
a point, which is output via "Point". If the user used a
"point-on-entity" option, the entity index is output via "idEntity".

Deprecated:
This API will not be updated anymore, please use ZwPointGetByPick()
See also
ZwPointGetByPick
Parameters
[in]Prompttext prompt
[in]Optpoint input option
[in]EmptyOk1 if <middle-click> without input is okay; else 0
[out]Pointpoint input
[out]idEntityentity id (input NULL to ignore)
Return values
succeeds
  • ZW_API_NO_ERROR
fail
  • ZW_API_GENERAL_ERROR : error or user-escape
  • ZW_API_INVALID_OUTPUT
