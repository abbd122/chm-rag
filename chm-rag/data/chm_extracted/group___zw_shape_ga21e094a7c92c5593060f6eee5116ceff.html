◆ ZwShapeEdgeSewAll()

ZW_API_C ezwErrors ZwShapeEdgeSewAll ( double  tolerance,
int *  openEdgesCount,
double *  maxGap 
)

#include <zwapi_shape.h>

Matches and sews free edges in the active part using the input proximity tolerance. If "tolerance" is input as 0.0, the active part's default proximity tolerance is used.

Note
Input NULL for "openEdgesCount" and/or "maxGap" if you are not interested in the information returned via those variables.
Parameters
[in]tolerance3D proximity tolerance (mm) (0.0 for default)
[out]openEdgesCountnumber of unmatched/open edges after sew (or NULL)
[out]maxGapmaximum gap between all matched edges after sew(or NULL)
Return values
succeeds
  • ZW_API_NO_ERROR
fail
  • ZW_API_GENERAL_ERROR
  • ZW_API_MODELING_DATA_ERROR
