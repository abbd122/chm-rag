ZW_API_C void cvxNewCommand ( void  )

#include <zwapi_cmd.h>

Displays buffered error messages and resets the GUI to prompt the user for a new command. Generally you would call this function at the end of your custom API code before returning control to ZW3D – unless your code is automatically called by a custom command template.
