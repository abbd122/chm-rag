◆ ZwPathSeparatorCheck()

ZW_API_C ezwErrors ZwPathSeparatorCheck ( const char *  sourcePath,
int  nBytes,
char *  path 
)

#include <zwapi_file_path.h>

Check the input path to see if it is properly formatted for the target OS. Forward or backward slashes are replaced with the appropriate path delimiter for the operating system. D:/aa/bb <==> D:\aa\bb Besides, this function will get the real path for the sourcePath, that means it will remove the path mark like '..' or '.' and translate to the absolute path.

Parameters
[in]sourcePathpath to a file or directory
[in]nBytesmax size of path
[out]pathcorrect path of sourcePath

fail

  • ZW_API_INVALID_PATH
  • ZW_API_INVALID_OUTPUT
