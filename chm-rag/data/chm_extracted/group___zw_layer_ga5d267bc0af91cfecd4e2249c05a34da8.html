#include <zwapi_layer.h>

Modifying the visibility state of the layer category with the specified name in the layer manager. A change in the visibility of a layer category changes the visibility of layers with filters for that category added. use ZW_CATEGORY_NO_CATEGORY can modify the visibility of layers with no category filter set.
