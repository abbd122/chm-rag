ZW_API_C evxErrors cvxPartPublishSet ( const vxName  Name,
int  Count,
const int *  idEnts,

#include <zwapi_cmd_dataexchange.h>

Use the function to create a publication set to publish the specified entity.

See also
cvxPartInqPublishSets(): get a list of indices of the publication set.
cvxPartInqPublishSet(): query data of the specified publication set.
cvxPartPublishSetDel(): delete the specified publication sets.
Parameters
[in]Namename of publication set (NULL to use default name)
[in]Countnumber of entities to publish
[in]idEntslist of entity ids to publish (shape, face, edge, curve, point, sketch, datum plane, datum axis, datum CSYS)
[out]idPubSetid of new publication set (NULL to ignore)
