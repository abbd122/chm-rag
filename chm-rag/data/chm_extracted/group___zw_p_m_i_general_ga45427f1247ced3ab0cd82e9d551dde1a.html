ZW_API_C ezwErrors ZwPMIDimensionFeatureControlCreateByText ( const char *  text,
szwPointOnEntity *  leaderPoint,

#include <zwapi_pmi.h>

Creates a feature control symbol (FCS) in active part or assembly. Function Guide: PMI -> Annotation -> Feature Control

Note
The caller is responsible to free the FCS by using ZwEntityHandleFree().
Parameters
[in]textthe FCS text
[in]leaderPointthe leader point, must be a point on the entity (input NULL to ignore)
[in]locationthe location point
[in]planethe placement plane of the FCS, must be a datum or planar face (input NULL to ignore)
[out]FCShandle of the newly created FCS (input NULL to ignore)
Return values
succeeds
