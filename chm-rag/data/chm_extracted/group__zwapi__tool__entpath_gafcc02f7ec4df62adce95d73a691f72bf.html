◆ cvxEntPathToRoot()

ZW_API_C evxErrors cvxEntPathToRoot ( svxEntPath *  EntPath)

#include <zwapi_tool_entpath.h>

Converts the input entity path from a path that is defined relative to the active target object to an "absolute" path that begins with the parent file of the target object. It is assumed that the entity path is an evaluated (i.e. non-persistent) entity path.
For example:
If the input EntPath's Id[0] = -1, such as Id = { -1, COMP, COMP, FACE};
the output EntPath's Id = { -2, PART, COMP, PART COMP, PART, FACE};

Parameters
[in,out]EntPathentity path
Return values
succeeds
  • ZW_API_NO_ERROR
fail
  • ZW_API_GENERAL_ERROR
  • ZW_API_INVALID_ENT_PATH
  • ZW_API_ENT_PATH_ERROR
//If you want to get current active pick path
svxEntPath path = {};
path.Count = 1;
path.Id[0] = -1;
cvxEntPathToRoot(&path);
