◆ cvxConfigSetNumber()

ZW_API_C evxErrors cvxConfigSetNumber ( const char *  Name,
double  Value 
)

#include <zwapi_config.h>

Sets the value of a setting from the Configuration form/config file. The variable value is assumed to be a number (0 is assigned to "Value" when the value is not a number).

Note
The input "Name" is defined in "zw3d installation dir\supp\config.xml.

[in]Namename of config file variable
[in]Valuevalue of the specified variable
