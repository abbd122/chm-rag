#include <zwapi_sketch_constraint.h>

Inquires the constraint status of the active sketch and retrieves the geometries.

Function Guide: Sketch->Settings->Constraint Status

See also
ZwSketchConstraintStatusDataFree
Note
The caller is responsible to free the geometryData by using ZwSketchConstraintStatusDataFree().
To get a correct result, it's recommended that to use this function with 'solveConstraints = 1'.
When the input parameter 'solveConstraints' is 1, this function will make the file be modified as the command 'Constraint Status' does.
Parameters
[in]solveConstraintswhether to solve the constraints before inquire the data, 1-yes, 0-no
[out]statusconstraint status, 1 for all geometries are well defined, else 0
[out]geometryDatageometry list with over defined, under defined and well defined constraints(NULL to ignore)
