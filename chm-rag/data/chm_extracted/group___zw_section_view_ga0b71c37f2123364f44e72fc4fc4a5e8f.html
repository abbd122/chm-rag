◆ ZwSectionViewDirectionReverse()

ZW_API_C ezwErrors ZwSectionViewDirectionReverse ( szwEntityHandle  sectionViewHandle)

#include <zwapi_view.h>

Reverse the direction of specific section views.

Note
This api only takes effect on 3 types of section view(for more detail please see ezwSectionViewMode), including ZW_SECTION_VIEW_SECTION_AT_PLANE, ZW_SECTION_VIEW_SECTION_WITH_ENVELOPE and ZW_SECTION_VIEW_SECTION_WITH_SLICE. You can use ZwSectionViewParameterGet to get type of section view.
Parameters
[in]sectionViewHandlehandles of the section view
Return values
succeeds

fail

  • ZW_API_WRONG_ROOT_ENV
  • ZW_API_INVALID_INPUT
  • ZW_API_INPUT_TYPE_ERROR
  • ZW_API_INVALID_OUTPUT
  • ZW_API_OUT_OF_MEMORY
