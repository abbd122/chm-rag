#include <zwapi_drawing_sheet.h>

Gets all dimensions in the specified drawing sheet, both visible and hidden.

Note

The caller is responsible to free the dimensions by using ZwEntityHandleListFree().
See also
ZwEntityBlankGet
Parameters
[in]sheetHandlesheet handle (input NULL to query active sheet)
[in]viewDimensionwhether to get dimensions associated to view (1: yes; 0: no)
[in]dimensionTypeCountnumber of dimension types (input 0 to query all dimensions)
[in]dimensionTypeListlist of dimension types (input NULL to query all dimensions)
[out]countnumber of dimensions
[out]dimensionslist of dimension handles
