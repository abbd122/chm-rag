◆ ZwFaceTextureAttributeGet()

ZW_API_C ezwErrors ZwFaceTextureAttributeGet ( szwEntityHandle  faceHandle,
szwTextureAttribute *  texture 
)

#include <zwapi_face.h>

Outputs texture attributes of the specified face.

Note
If a texture image file is embedded in a multi root file(Z3 file), it is automatically extracted to a "temp" sub-folder in the Z3 runtime folder. Path of the temp file is output via "texture->file". It is assumed the specified face resides in the active part.

[out]textureface texture attribute
