enum ezwGeometryCriticalPoint

#include <zwapi_util.h>

type of sketch geometry critical point

Enumerator
ZW_NO_POINT 

set the entity not the critical point of entity

ZW_FREE_POINT 

free point, non-critical point

ZW_START_POINT 

start point of curve or line (if the input is face, get point of (u.min, v.min))

ZW_END_POINT 

end point of curve or line (if the input is face, get point of (u.max, v.max))

ZW_CENTER_POINT 

center point, used only for arc or circle (if the input is face, get point of ((u.min+u.max)/2, (v.min+v.max)/2))

ZW_SPLINE_DEFINING_POINT 

control point, used only for splines

ZW_MIDDLE_POINT 

middle point of curve or line

ZW_ORIGIN_POINT 

origin of datum plane and datum CSYS
