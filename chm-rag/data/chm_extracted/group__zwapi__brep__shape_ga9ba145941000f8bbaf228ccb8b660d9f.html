◆ cvxPartInqShapeBox()

ZW_API_C evxErrors cvxPartInqShapeBox ( int  idShape,
svxMatrix *  Mat,
svxBndBox *  Box 
)

#include <zwapi_brep_shape.h>

Gets the bounding box of the specified shape in the active file. If "*Mat = NULL" && the target object class isn't VX_ROOT_PART(evxRootType), the bounding box coordinates are given relative to the global frame of the parent part.

If the target object class is VX_ROOT_PART && "*Mat = NULL", initializes "Mat" to the identity matrix. The "idShape" must belong to the active part, a bounding box relative to the specified local coordinate frame is output.

See also
cvxPartInqEntBox,cvxPartInqFaceBox,cvxEntBndBox. The three interfaces have similar functions.

[in]idShapeshape or face id
