#include <zwapi_drawing_general.h>

Outputs the handle of the assembly component associated with the specified entity in a 2D drawing view layout from a 3D assembly.

Assumes that "entityHandle" resides in the active target file.

Outputs "compHandle->innerData = nullptr" if no assembly component is found.

Outputs the names of the component's parent file and object (i.e. part) if non-NULL pointers are input to the "char *" variables for "file" and "parent".

[in]nFileBytesstring size of file name
