This function is used to create BOM table

Deprecated:
This API will not be updated anymore, please use ZwDrawingTableBOMCreate()
See also
ZwDrawingTableBOMCreate
Note
Interface cvxDwgTableBomInit() can initialize variables of this function.
After creating the BOM table, use interface cvxDwgTableInsert to insert the BOM table into the current drawing. VX_DWG_CON_USER_TEXT won't take effect in this Api.
// 1. Initialize settings
svxDwgBom bomData = {};
cvxDwgTableBomInit(&bomData);
//2. Create BOM table
...
int idBom = 0;
evxErrors err = cvxDwgTableBom(&bomData, &idBom);
//3. Insert BOM table to drawing
svxDwgTblInsert ins = {};
ins.idTable = idBom;
...
cvxDwgTableInsert(&ins);
