◆ cvxCompAddPathCons()

ZW_API_C evxErrors cvxCompAddPathCons ( const svxPntOnEntPath *  Pnt,
int  nPaths,
const svxEntPath *  Paths,
const svxPathConsData *  ConsData,
int *  idCons 
)

#include <zwapi_cmd_assembly.h>

Use the function to create a path constraint. If multiple curves or edges are input as constraint paths, all curves or edges are required to come from the same component and G1 continuous between curves or edges.

Parameters
[in]Pntpoint on component (edge, curve, face, datum plane, point)
[in]nPathsnumber of paths
[in]Pathspick-path list of paths to align to, the curves must be G1 continuous (edge, curve)
[in]ConsDataconstraint custom data (NULL to use default value)
[out]idConsindex of new constraint (NULL to ignore)
Return values
succeeds

fail
