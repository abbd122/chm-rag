ZW_API_C int cvxSrfFindPoles ( svxSurface *  srf,
double  tol,
int  poles[4] 
)

#include <zwapi_math_nurbssurface.h>

Deprecated:
This API will not be updated anymore,please use ZwFacePoleCheck()
See also
ZwFacePoleCheck

This routine determines which, if any, boundaries of srf are poles.
It sets the flags in the array poles appropriately.

Parameters
[in]srfpointer to NURBS surface data structure
[in]tolthe tolerance
[out]poles[4]poles[i] = 0 means the i-th boundary is not a pole.
poles[i] = 1 means it is a pole.
The boundaries are numbered counterclockwise, starting at
the bottom (v=0).
fPoles[0] -> pole at loV
fPoles[1] -> pole at hiU
fPoles[2] -> pole at hiV
fPoles[3] -> pole at loU
Return values
return
  • 0 - No poles found or error occurs
  • 1 - At least one pole found

  • ZW_API_GENERAL_ERROR
