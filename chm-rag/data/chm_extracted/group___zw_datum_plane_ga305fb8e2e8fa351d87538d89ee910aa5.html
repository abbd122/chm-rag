ZW_API_C ezwErrors ZwDatumPlaneCreateByOffsetPlane ( szwEntityHandle  originPlane,
szwPoint  origin,
double  offset,
szwEntityHandle *  plane

#include <zwapi_datum.h>

Create a datum plane object by a face.

For the input parameter 'origin', this function will calculate the projection point on the plane, then use it as the origin of the new plane.

Note
The memory referenced by "plane" is allocated by this function. The calling procedure is responsible to free it (ZwEntityHandleFree(&plane)).
