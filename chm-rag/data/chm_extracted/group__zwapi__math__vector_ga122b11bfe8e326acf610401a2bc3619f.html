ZW_API_C int cvxVecParallelChk ( int  Orient,
svxVector *  V1,
svxVector *  V2,
double  CosTol 
)

#include <zwapi_math_vector.h>

Determine whether two vectors are parallel within a specified angular tolerance.

The CosTol tolerance is the cosine of the maximum angle allowed between the
vectors for them to be considered parallel. The table below shows the maximum
angle allowed between the vectors (right) for several values of CosTol (left): 

                     CosTol               degrees
                     0.9          -        25.84
                     0.9848       -          10
                     0.9962       -          5
                     0.9986       -          3
                     0.9994       -          2
                     0.99985      -          1
                     0.99996      -         0.5
                     0.999998     -         0.1
Deprecated:
This API will not be updated anymore, please use ZwVectorParallelCheck()

ZwVectorParallelCheck
Parameters
[in]Orient0 = parallel vectors can be in same or opposite direction
1 = Parallel vectors must point in same direction
2 = Parallel vectors must point in opposite directions
[in]V1first vector
[in]V2second vector
[in]CosTolcosine of maximum angle allowed for parallelism
Return values
return
  • 1: vectors are parallel; if Orient is 1, the vectors also point in the same direction; if Orient is 2, the vectors point in opposite directions.
  • 0: vectors are NOT parallel, or are not oriented with respect to each other as required by Orient, or at least one vector degenerates to zero length, or error occurs.
