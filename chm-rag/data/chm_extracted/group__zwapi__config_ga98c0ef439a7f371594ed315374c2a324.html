◆ cvxConfigPathGet()

ZW_API_C evxErrors cvxConfigPathGet ( char *  path,
int  sizepath 
)

#include <zwapi_config.h>

Get config path for api developer. Api developers should save their config files here, if they have.

A path that is too long (longer than "sizepath") is truncated.

It is assumed that the input "path" is no longer than 512 bytes.

[in]sizepathmax size of path
