◆ cvxPartAddLineDim()

ZW_API_C evxErrors cvxPartAddLineDim ( int  idEnt1,
int  idEnt2,
const svxPoint *  TextPnt,
int  idPlane,
int *  idDim 
)

#include <zwapi_cmd_pmi.h>

Create a PMI linear dimension between the specified two entities in part. Supported entities include two points, a point and a line,two parallel lines, two parallel faces.

Parameters
[in]idEnt1id of the first entity
[in]idEnt2id of the second entity
[in]TextPntpoint coordinate of text placement point (NULL to ignore, automatic calculation)
[in]idPlaneid of insertion plane (datum or planar face), 0 to use default view plane
[out]idDimid of the dimension created (input NULL to ignore)
Return values
succeeds
  • ZW_API_NO_ERROR
fail
  • ZW_API_GENERAL_ERROR
  • ZW_API_INVALID_INPUT
  • ZW_API_ROOT_OBJ_ACT_FAIL
  • ZW_API_WRONG_ROOT_ENV
  • ZW_API_CMMD_INIT_ERROR
  • ZW_API_INPUT_ERROR
  • ZW_API_CMMD_EXEC_ERROR
