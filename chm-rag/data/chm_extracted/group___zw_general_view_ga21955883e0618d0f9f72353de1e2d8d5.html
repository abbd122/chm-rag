#include <zwapi_view.h>

Get the transform matrix between two view.

Note
If isReverse is 0, it means:transMat*sourceView matrix=destView matrix; else if isReverse is 1, it means:transformMatrix*destView matrix=sourceView matrix.
Parameters
[in]sourceViewthe handle of view
[in]destinationViewthe handle of view
[in]isReversereverse or not
[out]transformMatrixtransform matrix
