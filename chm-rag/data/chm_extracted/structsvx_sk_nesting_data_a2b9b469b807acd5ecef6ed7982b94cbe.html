vxLongName svxSkNestingData::reportDir

Specify save path and file name for nesting report.If not specified, the default save path and file name will be used.
for example: reportDir="D:\ZW3D\Nesting result.html".
"D:\ZW3D\" is the path where the file is saved. It can be changed according to the user's wishes. <br> "Nesting result" is name. It can be changed. <br> but,the reportDir must contain ".html",otherwise the interface cannot be used normally.
