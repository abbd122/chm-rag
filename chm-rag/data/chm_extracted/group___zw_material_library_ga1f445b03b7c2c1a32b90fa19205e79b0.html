◆ ZwMaterialLibraryCategoryDelete()

ZW_API_C ezwErrors ZwMaterialLibraryCategoryDelete ( const char *  libraryName,
const char *  categoryName 
)

#include <zwapi_materiallibrary.h>

Deletes the material category with the specified name. The function will only delete unused materials from the material category. Materials in use will not be deleted.
