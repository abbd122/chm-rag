#include <zwapi_sk3d_objs.h>

Outputs a curves list in specified 3d sketch .

Note
This function allocates memory for the out via "curves". And the calling procedure is responsible to use 'ZwEntityHandleListFree' for memory deallocating.
See also
ZwEntityHandleListFree.
Parameters
[in]sketchthe handle data of a specified 3d sketch entity
