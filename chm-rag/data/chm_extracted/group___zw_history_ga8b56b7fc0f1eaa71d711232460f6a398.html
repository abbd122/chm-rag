ZW_API_C ezwErrors ZwHistoryPaste ( const szwEntityHandle *  targetEntity)

#include <zwapi_history.h>

Paste operations currently on the clipboard into the history of the active part. It also works to paste component objects. If the 'targetEntity' is not NULL, it will paste the objects under the specified entity. What 'targetEntity' means? For example, if you have a file named A with the history objects a1, a2 and a3, and another file named B with the history objects b1 and b2, now we can clip the objects a2 and a3 by 'ZwHistoryClip' and paste them between b1 and b2 by 'ZwHistoryPaste' with set the 'targetEntity' to b1. And we can get the history objects of B as b1, a2, a3 and b2.

Parameters
[in]targetEntitythe target entity that pastes the objects under it, NULL to ignore
